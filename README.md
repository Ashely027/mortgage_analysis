# Texas Mortgage Analysis
## Project Overview
This project explores mortgage data in Texas to uncover trends, patterns, and insights in the housing finance sector. Using a combination of data cleaning, statistical analysis, and predictive modeling, the project aims to provide a comprehensive view of mortgage activity across the state.

## Data Source
The data used in this project is sourced from publicly available mortgage datasets, including the Home Mortgage Disclosure Act (HMDA) records for Texas. The dataset includes key variables such as loan amounts, interest rates, property locations, applicant demographics, and loan outcomes.

## Analysis
Key analyses performed in this project include:
- **Exploratory Data Analysis (EDA):** Understand distributions, detect anomalies, and visualize mortgage trends.
- **Data Cleaning & Wrangling:** Handle missing values, standardize formats, and prepare data for modeling.
- **Predictive Modeling:** Use machine learning models (e.g., XGBoost) to predict mortgage approval likelihood and loan amounts.
- **Insights:** Identify patterns by region, loan type, and applicant demographics.

## Technologies Used
- Python (pandas, numpy, scikit-learn, XGBoost, matplotlib, seaborn)
- Jupyter Notebook
- Git & GitHub for version control
